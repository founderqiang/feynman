# Figure Composer

Use this skill for multi-panel scientific figures.

Workflow:

1. State the figure's single claim and list every data artifact that supports it.
2. Draft a panel plan with labels, chart types, data sources, required annotations, and what each panel proves.
3. Use `figure-style` for each plot before composing the multi-panel layout.
4. Render, inspect, and revise for label collisions, axis readability, color meaning, caption fit, and data-source fidelity.
5. Save the final figure, source data, generation code, caption, and provenance sidecar.

Do not invent data or hide excluded data in summaries.
